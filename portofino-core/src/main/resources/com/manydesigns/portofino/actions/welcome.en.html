API

Your application exposes a REST API which depends on the resource-actions that it's made of. You can download the documentation of the REST API in OpenAPI (aka Swagger) format. This is machine-readable, so you can generate a client automatically in many programming languages from it.

Note that currently the OpenAPI documentation is cached, and if you change the structure of the application, you'll have to restart it in order to see the updated documentation.
