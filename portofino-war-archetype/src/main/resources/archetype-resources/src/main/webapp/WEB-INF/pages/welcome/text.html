Welcome to


You can use the wizard to create an application by analyzing an existing database schema. After that, you can modify the generated pages, move them around, and create new ones.

Have a look at the documentation, wiki, and mailing list to learn more: Portofino offers lots of features, the wizard is just an easy starting point! Don't be scared to run the wizard multiple times if you wish - it won't overwrite anything.

Also, take a tour of the samples to get a glimpse of what you can do with Portofino besides SCRUD (Search, Create, Read, Update, Delete data in a database).


You can edit this home page, and any other page, by logging in as admin/admin and clicking on the button in the top right corner.
